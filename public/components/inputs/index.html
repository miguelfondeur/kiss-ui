Inputs

Input components are used for forms.

Checkbox

Checkboxes are important for various reasons.

Example




HTML 
<form>
    <input type="checkbox" id="vehicle1" name="vehicle1" value="Bike">
    <label for="vehicle1"> I have a bike</label><br>
    <input type="checkbox" id="vehicle2" name="vehicle2" value="Car">
    <label for="vehicle2"> I have a car</label><br>
    <input type="checkbox" id="vehicle3" name="vehicle3" value="Boat">
    <label for="vehicle3"> I have a boat</label><br>
</form>

Radio Buttons

Radio buttons are used to select multiple values in a form.

Example



HTML

    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label><br>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label><br>
    <input type="radio" id="other" name="gender" value="other">
    <label for="other">Other</label>

Select Box

Select boxes are form elements that allow you to choose from a list of options.

Example

HTML 
<form>
    <label for="cars">Choose a car:</label>
    <select id="cars">
        <option value="volvo">Volvo</option>
        <option value="saab">Saab</option>
        <option value="opel">Opel</option>
        <option value="audi">Audi</option>
    </select>
</form>

Range Selector

The range input selector allows you to input a number within a specific programable range.

Example

    <input type="range" id="points" name="points" min="0" max="10">

Search with Datalist

The datalist input allows the user to select some predefined search options when using the search input.

Example

HTML 
<form>
    <label for="browser">Choose your browser from the list:</label>
    <input list="browsers" name="browser" id="browser">
    <datalist id="browsers">
        <option value="Edge">
        <option value="Firefox">
        <option value="Chrome">
        <option value="Opera">
        <option value="Safari">
    </datalist>
</form>

Search with Reset Button

The reset input button clears a search input when clicked. With the reset and datalist elements you can create a more user friendly search experience.

Example

HTML

    <input type="text" id="id" name="id" />
    <input type="reset" value="Reset">
