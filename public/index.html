Light UI

Keep it Short and Simple

Light is....

A collections of vanilla HTML, CSS, and Javascript patterns that takes full advantage of native browser features and API's.

Light is not....

Light is NOT a framework. There are no third party dependencies. It is not opinionated. Light is a starting point for your project that minimizes complexity for the developer while simultaneously priotitizing an accessible User Experience.
